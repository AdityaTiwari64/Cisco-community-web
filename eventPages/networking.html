    NETWORKING CHALLENGE - FULL DETAILS

    ADVANCED LEVEL

    Smart City Network Infrastructure Design

    Scenario: You are tasked with designing a comprehensive network infrastructure for a smart city project that will connect 10,000+ IoT devices, support autonomous vehicles, and provide high-speed internet to 500,000 residents.

    Requirements:

    • Design a scalable network topology supporting 99.99% uptime
    • Implement Quality of Service (QoS) for critical services
    • Ensure network security with zero-trust architecture
    • Support for 5G integration and edge computing
    • Disaster recovery and redundancy planning
    • Cost optimization while maintaining performance

    Deliverables:

    • Network topology diagram with detailed specifications
    • Security implementation plan
    • Performance analysis and capacity planning
    • Cost breakdown and ROI analysis
    • Implementation timeline and risk assessment

    Evaluation Criteria:

    • Technical feasibility and innovation (30%)
    • Scalability and performance optimization (25%)
    • Security implementation (20%)
    • Cost-effectiveness (15%)
    • Presentation and documentation (10%)
    Duration: 48 Hours
    Team Size: 2-4 Members
    Prize Pool: ₹50,000
