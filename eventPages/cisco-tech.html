    CISCO TECH CHALLENGE - FULL DETAILS

    ADVANCED LEVEL

    Software-Defined Network (SDN) Solution

    Scenario: Implement a comprehensive Software-Defined Network (SDN) solution with advanced traffic management, Quality of Service (QoS), and network automation for a large enterprise with multiple data centers.

    Requirements:

    • SDN controller implementation with OpenFlow protocol
    • Dynamic traffic management and load balancing
    • Advanced QoS policies for different application types
    • Network function virtualization (NFV) integration
    • Automated network provisioning and configuration
    • Network monitoring and analytics dashboard

    Deliverables:

    • SDN architecture design and implementation
    • Traffic management algorithms and QoS policies
    • Network automation scripts and orchestration
    • Performance monitoring and analytics system
    • Network security implementation and testing

    Evaluation Criteria:

    • SDN implementation and functionality (35%)
    • Traffic management and QoS effectiveness (25%)
    • Network automation and orchestration (20%)
    • Performance optimization (15%)
    • Security and monitoring capabilities (5%)
    Duration: 48 Hours
    Team Size: 3-4 Members
    Prize Pool: ₹55,000
