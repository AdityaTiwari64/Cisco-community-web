    CYBERSECURITY CHALLENGE - FULL DETAILS

    ADVANCED LEVEL

    The "Phishing Shield"

    Description: : An intelligent tool that uses AI to detect sophisticated phishing emails that traditional spam filters miss. It solves the problem of increasingly clever phishing attacks by analyzing email context for unusual activity, not just bad links. The system can also "virtually" visit a suspicious link in a secure sandbox to analyze the landing page for signs of a scam. It then warns the user with a simple, color-coded alert (Green, Yellow, Red) that is easy to understand.

    Key Requirements:

    • AI/ML model for natural language processing (NLP) to analyze email text and context
    • A secure, isolated sandbox environment to safely render and analyze web pages from suspicious links.
    • Integration with common email clients (e.g., as a browser extension for Gmail or an Outlook add-in).
    • A simple and intuitive user interface for displaying threat levels (e.g., color-coded alerts).
    • Real-time URL scanning and analysis capabilities.

    Outcomes:

    • A significant reduction in successful phishing attacks for users.
    • Increased user awareness and ability to recognize phishing attempts.
    • A measurable accuracy rate (e.g., >95%) in correctly identifying phishing emails while minimizing false positives.

    Condition:

    • The system must have access to email content for analysis, which requires user consent and strong privacy safeguards.
    • The sandboxing environment must be completely isolated to prevent any potential malware from affecting the user's system
    • The model must be continuously updated to recognize new and evolving phishing techniques

    Potential Datasets:

    • PhishTank: A collaborative clearinghouse for data and information about phishing on the Internet. Link to Phishtank
    • Kaggle Phishing Email Dataset: A collection of emails labeled as phishing or legitimate. Link on Kaggle
    • The model must be continuously updated to recognize new and evolving phishing techniques. Link to UNB Dataset
    Duration: 36 Hours
    Team Size: 1 Member
    Prize Pool: ₹60,000
