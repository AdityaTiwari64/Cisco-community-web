    IOT CHALLENGE - FULL DETAILS

    INTERMEDIATE LEVEL

    The "Signal-Sense" Network Mapper

    Description: A solution to the common problem of unreliable campus Wi-Fi and cellular service. This tool is a crowdsourcing app that allows students to map network dead zones in real-time. The app collects anonymous data on signal strength, speed, and latency at different locations and times. The AI component uses this data to create a predictive heatmap of the campus network, forecasting when and where congestion will occur. This allows students to proactively find the best spots for stable connectivity.

    Requirements:

    • Integration of 20+ IoT sensors (temperature, humidity, motion, light, etc.)
    • Predictive analytics for energy optimization (30% reduction target)
    • Real-time security monitoring with facial recognition
    • Voice and mobile app control interfaces
    • Machine learning for behavior pattern recognition
    • Edge computing for low-latency responses

    Deliverables:

    • IoT system architecture and sensor deployment plan
    • Working prototype with key functionalities
    • Mobile application for system control
    • Energy optimization algorithm and results
    • Security implementation and testing report

    Evaluation Criteria:

    • System functionality and innovation (35%)
    • Energy optimization effectiveness (25%)
    • User interface and experience (20%)
    • Security implementation (15%)
    • Technical documentation (5%)
    Duration: 36 Hours
    Team Size: 2-4 Members
    Prize Pool: ₹40,000
