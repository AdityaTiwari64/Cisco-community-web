    CLOUD COMPUTING CHALLENGE - FULL DETAILS

    ADVANCED LEVEL

    Multi-Cloud Disaster Recovery Solution

    Scenario: Design and implement a comprehensive multi-cloud disaster recovery solution for a financial services company that processes 1 million transactions daily across AWS, Azure, and Google Cloud platforms.

    Requirements:

    • Automated failover with RTO < 15 minutes and RPO < 5 minutes
    • Cross-cloud data synchronization and consistency
    • Cost optimization across multiple cloud providers
    • Compliance with financial regulations (PCI DSS, SOX)
    • Real-time monitoring and alerting system
    • Automated testing of disaster recovery procedures

    Deliverables:

    • Multi-cloud architecture diagram and implementation plan
    • Automated deployment scripts and Infrastructure as Code
    • Disaster recovery runbook and testing procedures
    • Cost analysis and optimization recommendations
    • Security and compliance assessment report

    Evaluation Criteria:

    • Architecture design and scalability (30%)
    • Automation and reliability (25%)
    • Security and compliance (20%)
    • Cost optimization (15%)
    • Innovation and best practices (10%)
    Duration: 48 Hours
    Team Size: 2-4 Members
    Prize Pool: ₹50,000
